# Colliding widgets

This directory contains the code for a Qt widgets application which shows two
movable widgets on the main form. Those two widgets (initially shown as green
rectangles) can be moved by pressing the left mouse button and holding it
while moving the mouse.

The moved wiget changes its colour when it collides (i. e. overlaps) with the
other widget and changes back to the original colour when it no longer overlaps
with the other widget.
