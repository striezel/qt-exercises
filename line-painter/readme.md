# Line painter

The code in this directory builds a simple drawing application that can be used
to draw lines inside the main window's central widget. Pressing the left mouse
button on a free spot on the main window and moving the mouse inside the window
will draw a line that follows the mouse movements. Upon releasing the left mouse
button the line is finished and the process can be repeated, drawing another
line.

Line colour and line thickness can also be changed via the application's menu,
but only for the latest line. So make sure the colour and thickness are to your
liking before starting a new line.

The drawing can also be saved to an image file (PNG).

Finally, you can also delete the drawing of lines and start on a fresh "canvas".

Note: Currently, the user interface uses German text, so you may not be able to
follow the menu intuitively, if you cannot read German.
